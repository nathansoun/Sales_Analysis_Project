# Sales_Analysis_Project
Retail sales analytics project using Excel to analyze sales trends, KPIs, and customer behavior.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Formulas & Conditional Formatting

---

## Dataset
- 1000 retail sales records
- 3 product departments
- Transaction and customer demographic data

---

## Dashboard Features
- Interactive slicers for dynamic filtering
- Real-time KPI tracking
- Sales trend visualizations
- Department performance analysis
- Customer purchasing insights

---

## Key KPIs
The dashboard automatically updates based on slicer selections and displays:
- **Total Sales**
- **Average Transaction Value**
- **Top Performing Category**

---

## Business Questions Answered
- Which product categories generate the most revenue?
- How do sales trends change over time?
- How does customer purchasing behavior vary across categories?

---

## Key Insights
- Certain product categories consistently outperformed others in total revenue
- Average transaction values increased during peak shopping periods like holidays
- Interactive filtering revealed differences in department performance across time periods and customer groups
